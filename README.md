# Drag and drop Vanilla JS from Typescript

This project is only to practice Vanilla js with Typescript

## Installation

```js
npm install
```

## Usage

1. **Step** Typescript run
```bash
tsc -w
```
in this project's root
2. **Step** Server run 
```bash
npm start
```
